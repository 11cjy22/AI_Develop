## 📞 常见问题

### Q1: 训练时间很长怎么办？
A: 可以减少 epochs（改为50），或使用GPU加速。

### Q2: 数据下载失败？
A: 检查网络连接，或使用已有的CSV文件。

### Q3: 模型性能不理想？
A: 尝试调整：
- 学习率（0.001 → 0.0005）
- Batch size（32 → 64）
- 窗口大小（30 → 60）

### Q4: 前端显示异常？
A: 确保安装了最新版本的 streamlit 和 plotly：
```bash
pip install --upgrade streamlit plotly
```
