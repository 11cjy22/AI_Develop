## 🚀 快速开始

### 方法1: 一键启动（推荐）
```bash
./start.sh
```

### 方法2: 手动执行
```bash
# 1. 安装依赖
pip3 install -r requirements_improved.txt

# 2. 下载数据
python3 data_download_improved.py

# 3. 处理数据
python3 data_process_improved.py

# 4. 训练模型
python3 train_improved.py

# 5. 启动前端
streamlit run app_beautiful.py
```

## 🔧 故障排查

### 问题1: 模块未找到
```bash
pip3 install -r requirements_improved.txt
```

### 问题2: 数据下载失败
- 检查网络连接
- 或使用已有的CSV文件

### 问题3: 训练过程中断
- 减少epochs（改为50）
- 或使用已训练好的模型

### 问题4: 前端无法启动
```bash
streamlit run app_beautiful.py --server.port 8502
```
