# 📈 A 股 + 黄金市场风险预测系统

基于深度学习的 A 股与黄金联动风险预测系统，使用先进的 CNN-LSTM+Attention 混合模型，根据过去 30 天历史数据预测下一天是否出现大于 1.5% 的大跌风险，并提供专业化的可视化界面。

## 🎯 项目概述

这是一个专业的金融科技项目，结合了现代深度学习技术和传统金融分析，旨在为投资者提供市场风险预警。系统通过分析股票市场（沪深300等主要指数）和黄金市场的联动效应，实现更准确的风险预测。

## ✨ 核心功能

- **📊 多源数据获取**：使用 akshare 实时下载沪深300、上证指数、深证成指、创业板指和黄金市场数据
- **🔧 专业数据处理**：完整的数据清洗、特征工程、技术指标计算和数据标准化流程
- **🧠 智能模型训练**：基于 CNN-LSTM+Attention 混合架构的深度学习模型
- **⚡ 实时风险预测**：预测下一天市场大跌风险（跌幅 > 1.5%）
- **🎨 可视化界面**：基于 Streamlit 开发的现代化 Web 应用界面
- **📈 技术分析展示**：实时展示价格走势、技术指标和预测结果

## 🧠 模型架构

### CNN-LSTM+Attention 混合模型

#### 1. CNN 部分（特征提取）
- 2层 Conv1D + MaxPooling1D
- 提取短期波动模式和局部特征
- 滤波器数量：64 → 128

#### 2. LSTM 部分（时序建模）
- 2层 LSTM 网络
- 学习时间序列的长期依赖关系
- 隐藏单元：128 → 64

#### 3. Attention 机制（重点识别）
- 自注意力机制
- 自动关注对预测最重要的时间步
- 提高模型解释性

#### 4. 全连接层（决策输出）
- Dense(64) + Dropout(0.5)
- Dense(32)
- 输出层：Sigmoid 激活（二分类）

## 🔧 特征工程

### 技术指标计算
- **MACD**：指数平滑移动平均线
- **KDJ**：随机指标
- **RSI**：相对强弱指标  
- **布林带**：波动率指标
- **移动平均线**：MA5, MA20
- **波动率**：20日标准差

### 数据标准化
- 使用 StandardScaler 进行标准化
- 更适合金融时间序列数据
- 保存标准化器用于预测时的一致性

## 📊 预测功能

### 风险定义
- **风险信号**：次日跌幅 > 1.5% → label=1
- **正常信号**：跌幅 ≤ 1.5% → label=0

### 预测输出
- **风险概率**：模型预测的风险概率值
- **预测结论**："有风险" 或 "安全"
- **投资建议**：基于预测结果的具体建议
- **原因分析**：多角度的风险原因解释
